## Getting Started ðŸ¤©ðŸ¤—:

- Fork this repo (button on top)
- Clone on your local machine

```terminal
git clone (URL)
```
- Navigate to project directory.
```terminal
cd hacktoberfest2022
```

```terminal
git init
```

- Have a look on issue section do work on that comment down assign me in particular issues

```terminal
git status
```
- Red text means file added but not added in github repo

```markdown
git add .
```
```terminal
git status
```
- Green text means file added in github repo
- Commit your changes.

```markdown
git commit -m "Relevant message"
```
- Then push 
```markdown
git push 
```
```terminal
git status
```

- Create a new pull request from your forked repository
